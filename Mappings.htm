ColdFusion Mappings

 
 

ColdFusion enables you to create logical aliases for physical directories on your server.

  • For Logical Path, enter the path relative to the root URL, such as /images.
  • For Directory Path, browse to the corresponding absolute path, such as C:/images.
  • To edit or delete a logical path, select it from the list below.
